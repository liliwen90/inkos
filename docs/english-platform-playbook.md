## 二、Royal Road — 最高优先级 ⭐⭐⭐

### 2.1 为什么是第一站
- **西方网文的起点中文网**：LitRPG/Progression Fantasy 最大聚集地
- **我们的 genre 完美匹配**：litrpg、system-apocalypse 已在 HintOS 配好
- **AI 内容友好**：允许 AI-Generated，只需打标签
- **零门槛起步**：免费注册，无国籍限制
- **自带变现体系**：读者打赏（Reader Support）+ 章节 Stub

### 2.2 注册流程
1. 访问 `royalroad.com` → 点击 "Create an account"
2. 填写：用户名 / 邮箱 / 密码（需 13 岁以上）
3. 邮箱验证 → 完成
4. **无需真实姓名**，用笔名即可
5. 可选升级 Author Premium ($59.99/年)：
   - 详细数据分析
   - 评论推送通知
   - 自定义页头横幅
   - EPUB 导出
   - 查找替换
   - 自动草稿保存
   - 章节导入

### 2.3 发布流程
1. 登录 → 顶部 "Write" → Author Dashboard
2. 点击 "Add New" → New Fiction Submission
3. 填写：
   - **标题**（不能含脏话/侵权内容）
   - **简介**（≥50 字符）
   - **封面**（400x600px，可选）
   - **Genre**（1-4 个，如 Fantasy / Sci-fi）
   - **Tags**（如 LitRPG、Male Lead、System、Apocalyptic）
   - **内容警告**（暴力/脏话/性暗示/AI标签）
   - **第一章内容**（提交时需附带第一章）
4. 提交后进入**人工审核**：
   - 审核内容：抄袭检查、标签合规、敏感内容、AI标注
   - 通常 **12-24 小时** 通过
   - 如果在其他平台发过，需要**所有权验证**：在源站描述里加一行 "I will be posting this on RoyalRoad" → 提 support ticket
   - **约 10% 投稿被拒**（抄袭/标签错误为主）
5. 审核通过后 → 持续上传新章节，无需再审

### 2.4 AI 内容政策（关键！）
Royal Road 2023-06 发布正式 AI 文本政策，三级分类：
1. **General Assistive** (Grammarly 等校对) → 无需任何标签
2. **AI-Assisted** (人写内容 + AI 润色/改写) → 需打 `AI-Assisted` 标签
3. **AI-Generated** (AI 生成 + 人类引导编辑) → 需打 `AI-Generated` 标签

**HintOS 策略**：
- HintOS 管线生成属于第 3 类（AI-Generated）
- **必须**打 `AI-Generated` 标签 + 在简介说明
- 好消息：RR 明确表示**不会因为 AI 标签歧视排名**，靠质量说话
- 他们承认 AI 检测不可靠（最高准确率仅 84%），不搞检测封号
- **但质量必须高**：低质量 AI 内容会被举报删除
- HintOS 的 26 维审计 + AIGC 检测 + 5 模式修订是核心竞争力

### 2.5 变现路径

#### 路径 A：读者打赏 (Reader Support)
- 读者可以通过 **BlueSnap** 支付渠道向作者打赏
- 作者需注册 **Tipalti** 账户才能接收付款
- Tipalti 要求：身份验证 + 税表（非美国人填 W-8BEN）
- 最低起付额由 Tipalti 设定（通常 ~$25-50）
- **每月发放一次**，余额达到最低阈值后自动打款
- 支付方式：PayPal / 银行转账 / eCheck
- ⚠️ OFAC 制裁国家不可用（但中国不在制裁名单内）

#### 路径 B：Stub + Amazon KDP（核心变现）
- **Stub** = 前 N 章节免费发 RR → 后续章节下架 → 打包成 eBook 在 Amazon 卖
- 这是 RR 顶流作者的主要收入来源
- 典型操作：RR 免费 50 章养粉 → Volume 1 上 Kindle → 删 RR 对应章节 → RR 引流 Amazon
- RR 明确允许 Stub，只是链接规则：广告里不能直链到 Stubbed 小说的第一章

#### 路径 C：Patreon 高级章节
- 在 RR 免费连载 → Patreon 上提前 10-20 章
- 读者为 "抢先看" 付费
- 这是 **RR + Patreon 联动** 的经典模式
- 注意：RR 简介里**不能放** Patreon 捐赠链接（限 fanfiction 规则）
- 但可以在个人主页和章节作者备注里放

#### 路径 D：RR 站内广告
- 有一定读者后，可以买 RR Leaderboard/Rectangle 广告推广
- 价格实惠：$55 起（14-28万次展示）
- 最大包：$275 / 85-170万次展示
- **仅限推广 RR 站内小说或 Amazon 链接**

### 2.6 提现闭环
```
注册 RR → 发布小说 → 获得读者 
                     ↓
             RR Reader Support (BlueSnap收款)
             Patreon 订阅收入
             Amazon KDP 销售收入
                     ↓
           Tipalti (RR打赏) / PayPal (Patreon) / 银行转账 (KDP)
                     ↓
           中国银行账户 或 PayPal 提现
```

## 三、ScribbleHub — 第二站 (同步发) ⭐⭐

### 3.1 定位
- 偏 ACG/日系的英文轻小说站
- GL（百合）、异世界等二次元题材特别强
- 社区氛围更 casual，审核宽松
- **没有站内变现手段**，纯流量+外链

### 3.2 注册发布
1. `scribblehub.com` → Register
2. 邮箱/用户名/密码
3. Author Dashboard → New Series
4. 填写标题/简介/标签/封面
5. 审核快速（通常几小时内）
6. AI 内容：无强制标签要求，建议自行标注避免社区反感

### 3.3 变现
- **无站内变现**
- 唯一方式：个人主页放 **Patreon / Ko-fi / PayPal.me** 链接
- 本质上是一个**流量获取渠道**，引流到 Patreon/Amazon

### 3.4 HintOS 策略
- **同步发**到 ScribbleHub（和 RR 相同内容）
- 在 profile 里放 Patreon 链接
- 增加一个曝光渠道，零成本运维

## 四、Kindle Direct Publishing (KDP) — 变现终极站 ⭐⭐⭐

### 4.1 定位
- Amazon 自出版平台，全球最大电子书市场
- 几乎所有 RR 顶流最终走向 KDP
- **Kindle Unlimited (KU)** 的页读分成是核心收入

### 4.2 注册流程
1. `kdp.amazon.com` → Sign Up
2. 需要：
   - Amazon 账户
   - **真实姓名 + 地址**
   - **税务信息**：非美国人需填 **W-8BEN**（证明不是美国纳税人）
   - **银行账户信息**（用于收款）
3. 税务面谈(Tax Interview)完成后开通发布权限
4. ⚠️ 中国地址/银行可用，Amazon 支持中国作者

### 4.3 发布流程
1. KDP Dashboard → Create New Title → Kindle eBook
2. 填写：
   - **语言：English**
   - **书名/副标题/系列名**（如 "The Jade Protocol: A LitRPG Adventure (Book 1)"）
   - **作者名**（笔名即可）
   - **简介 / 关键词（7个） / 分类（2个）**
   - **上传封面**（推荐 2560x1600px，2:3 比例）
   - **上传正文**（支持 EPUB / Word .docx）
3. 定价：
   - **$2.99-$9.99** → 70% 版税
   - 其他价位 → 35% 版税
   - 建议：Volume 1 定 **$4.99**，后续 $4.99-$6.99
4. **Kindle Unlimited (KU) 注册**：
   - 勾选 "KDP Select" → 加入 KU
   - ⚠️ **KDP Select 独占**：加入后该书只能在 Amazon 卖（不能同时在 RR 卖完整版）
   - 这就是为什么要 "Stub" — RR 发部分免费章 → 后续章上 KDP
5. 审核：自动审核，通常 **72 小时内** 上架

### 4.4 变现
| 收入来源 | 机制 | 估算 |
|---------|------|------|
| 电子书销售 | 70% 版税 (定价$2.99-$9.99) | 每本 $3.50 (定$4.99) |
| KU 页读 | 按实际阅读页数分成 KENPC | ~$0.004-0.005/页 |
| 平装本 | 按需印刷 (POD) | 利润低，$1-3/本 |

### 4.5 提现
- **每月自动打款**（月结 60 天后）
- 最低提现：**$100**
- 支持**银行电汇**到中国银行账户（需 SWIFT 代码）
- 也支持支票（不推荐，慢且贵）
- ⚠️ 注意：Amazon 会代扣美国预扣税（有W-8BEN可按中美税收协定降至 **10%**）

### 4.6 AI 政策
- 2023 年起要求发布时声明是否使用 AI 辅助
- 选项："使用 AI 生成" vs "使用 AI 辅助"
- 不会因 AI 标签拒绝发布
- 但如果被举报为纯 AI 低质量内容，可能被下架

## 五、Kindle Vella — 暂缓 ⭐

### 5.1 为什么暂缓
- **仅限美国居民/纳税人**（需 SSN 或 ITIN）
- 中国作者**无法直接注册**
- 市场还在培育中，读者基数远不如 RR

### 5.2 若未来需要
- 需要美国身份（或通过 LLC 注册）
- 按集/章节发布，读者用 Token 解锁
- 作者获得 50% Token 收入
- 前 3 集免费 → 后续付费
- 最低提现 $10

### 5.3 结论
除非获得美国身份/ITIN，否则跳过。RR + KDP 已覆盖同类需求。

## 六、Webnovel (Qidian International) — 观望 ⭐

### 6.1 定位
- 阅文集团旗下，面向全球的网文平台
- 翻译中文小说 + 接收英文原创
- **合同制**：签约后按字数/解锁分成

### 6.2 问题
- **签约合同争议大**：早期合同被批为"版权掠夺"（虽然后来改善）
- 签约后**版权归平台**（至少全球数字版权）
- AI 内容政策不透明，签约编辑可能拒绝 AI 稿
- 分成比例不如 RR+KDP 组合

### 6.3 结论
- 如果内容很适合翻译受众（东方幻想英文化），可以试投
- 但**不建议签约独家**，因为版权风险
- 优先级远低于 RR + KDP

## 七、Patreon — 变现加速器 ⭐⭐⭐

### 7.1 角色
不是发布平台，是**变现放大器**。与 RR 连载联动使用。

### 7.2 注册流程
1. `patreon.com` → "Create on Patreon"
2. 创建 Creator 页面
3. 设置 Tier（层级）：
   - **$3/month** — 提前 5 章
   - **$5/month** — 提前 10 章 + 作者笔记
   - **$10/month** — 提前 20 章 + 大纲投票 + Discord VIP
4. 连接 **Stripe** 或 **PayPal** 收款
5. 设置 Goals + About 页面
6. 即时生效，无审核

### 7.3 变现
- 平台抽成：8%（Pro Plan）或 12%（Premium Plan）
- 支付处理费另算：~2.9% + $0.30/笔
- **实际到手约 85-88%**
- 最低提现：**$1**（但处理费不划算，建议积累到 $25+）

### 7.4 提现
- **Stripe** → 直接打到银行账户（支持中国银行）
- **PayPal** → PayPal 余额 → 提现到银行
- 每月自动打款（1 号处理，3-5 工作日到账）

### 7.5 最佳实践
- RR 正文免费连载 → Patreon 超前章节
- 每周更新 3-5 章（免费端），Patreon 始终提前 10-20 章
- 月入预估：
  - 50 个 $5 patron = **$250/month** (扣费后 ~$213)
  - 200 个 $5 patron = **$1000/month** (扣费后 ~$850)
  - 头部 RR 作者 Patreon 月入 $5,000-$20,000+

## 九、税务与合规备忘

### 中国公民发布英文小说的税务要点

| 平台 | 税表要求 | 预扣税率 | 中美税收协定 |
|------|---------|---------|------------|
| KDP | W-8BEN | 30% → **10%**（有协定） | ✅ |
| Patreon | W-8BEN | 30% → **0%**（服务收入） | ✅ |
| Royal Road (Tipalti) | 取决于 Tipalti 要求 | 视情况 | ✅ |

- **W-8BEN** = 非美国人税务身份声明表（在各平台税务面谈中在线填写）
- 填 ITIN（有的话）或 Foreign TIN（中国身份证号）
- 中美税收协定使版税预扣从 30% 降至 10%（KDP）
- Patreon 收入按"服务收入"算，预扣可能为 0%
- 中国端：按"稿酬所得"报税（年收入 ≤ 4万免申报，但建议正规申报）
