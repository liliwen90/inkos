## 题材禁忌

- 主角靠天赋碾压一切不需要训练
- 比赛结果全靠运气或对手失误
- 队友只是主角的陪衬工具人
- 体育项目的规则随剧情需要修改
- 伤病说好就好没有恢复过程
- 对手全是坏人，赢了就道歉跪地
- 教练只会喊加油没有战术

## 竞技规则

- 体育项目的规则、术语必须准确
- 训练方法和周期要有运动科学基础
- 技术提升需要时间积累，不可一夜大成
- 伤病有合理的恢复周期和后遗风险
- 比赛中的体力消耗是真实的约束
- 天赋可以是优势，但磨练和战术才是胜负手
- 同一场比赛中的战术调整要有逻辑依据

## 语言铁律

- 比赛描写用运动员视角的体感，不用解说员视角的全知转播
- ✗"他一个漂亮的转身过人！全场沸腾！" → ✓"球触到脚面的一瞬间他知道方向对了——后卫的气息从左耳消失"
- 技术动作要有身体感：重心转移、肌肉发力、呼吸节奏
- ✗"他投出了一个完美的三分球" → ✓"手指离开球的时候旋转是对的，抛物线的最高点他就知道进了"
- 队友之间的交流用眼神、手势、跑位意图传递，不用赛场上大段对话
- 失败的描写和胜利一样重要——输球后的更衣室比赛后庆祝更有深度

## 叙事指导

竞技体育的核心是"有规则的较量"和"突破自我的极限"。
比赛不只是比谁强——心理、状态、战术、临场应变都是变量。

训练章不是灌水——训练中的细节、痛苦、瓶颈是成长的证据。
队友关系通过训练磨合和比赛默契建立，不靠口号和宣言。
教练的价值体现在战术设计、心理疏导和关键时刻的决策。

对手是另一面的主角，值得尊重和完整刻画。
逆风翻盘需要铺垫——对手的弱点、己方的潜力、战术的调整。
不是每场比赛都能赢，输球是成长的必要组成部分。

## 对话铁律

- 比赛中的对话要短：喘息间的一两个字，不是完整的句子
- 教练的战术指导要具体：✗"你要更努力！" → ✓"手腕再压低半寸，小臂别外翻"
- 队友之间的磨合用场上默契体现，不用“我们是最棒的团队”式口号
- 赛后采访和更衣室的对话气氛完全不同——公开场合套话，私下场合真话
- 对手之间的尊重用行为而不是台词：一个点头、一个拍肩、等在通道口

## 反AI写作模式

- 禁止用解说员视角写比赛：✗"竟然完成了一次完美的三分！" → ✓用球员的手感、覆觉、听觉写
- 禁止训练章流水账：✗"他练了三组爆发力，然后跑了5000米" → ✓只写某个训练细节的突破时刻
- 禁止“他感受到了竞技的魅力”式心理总结——竞技魅力通过细节传达
- 比赛结果不能每次都在最后一秒绝杀——有时是大比分的抗贫、有时是全场压制

## 章节钩子

- 比赛章结尾：赢了但核心球员受伤 / 输了但对手说了一句让人思考的话 / 下一轮的对手是曨日队友
- 训练章结尾：突破了瓶颈但代价是旧伤复发 / 教练变更 / 被调整到替补席
- 团队章结尾：队友之间的裂缝浮现 / 转会消息 / 某人的身体数据出了问题
