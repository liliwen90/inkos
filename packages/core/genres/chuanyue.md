## 题材禁忌

- 穿越者无所不知无所不能，现代知识=全能百科
- 古代/异界人毫无抵抗力全员跪拜
- 现代技术在异世界无条件完美复刻（没有原材料问题）
- 穿越者完全不需要适应期，开局就如鱼得水
- 身份伪装永远不穿帮，言行举止毫无破绽
- 对原住民文化和价值观全面否定（"你们都是愚昧的"）
- 穿越理由一笔带过后再也不提，像天然就属于这个世界

## 穿越约束

- 穿越者的知识优势必须有边界：不是所有现代知识都记得，更不是都能用
- 语言屏障是真实的：穿越古代不可能开口就说文言文，异界不可能自带翻译
- 技术移植受限于当地原材料、工具和人力水平（想造火药先得找到硝石和硫磺）
- 穿越者的心理适应需要过程：文化冲击、道德冲突、孤独感
- 蝴蝶效应必须体现：改变历史进程会产生连锁反应，不可能只改好的不改坏的
- 身份伪装有破绽风险：饮食习惯、卫生习惯、常识盲区都可能露馅
- 现代人不是天生的领导者/战略家——光知道历史走向不等于有执行力

## 语言铁律

- 穿越者的内心独白可以用现代口语，但对外说话必须适配环境
- ✗"这不就是清朝版的供应链管理吗" → ✓"他在心里把这叫供应链，嘴上说的是'转运之法'"
- 先知优势通过具体行为兑现，不通过独白式自夸："我知道接下来会发生什么"
- ✗"他知道三年后这里会发生大地震，所以决定提前布局" → ✓"他买下了那片没人要的荒地。三年，他等得起"
- 土著不是NPC——他们有自己的智慧、经验和判断，不会因为穿越者说了一句话就彻底信服
- 穿越者偶尔的"口误"（现代词汇/概念泄露）是好的叙事工具

## 叙事指导

穿越的核心吸引力是"信息差"——穿越者知道别人不知道的。
信息差的兑现必须有策略：什么时候用、怎么用、用了之后的连锁反应。

适应期是穿越文最有戏剧张力的阶段，不要跳过。
穿越者的第一个月比后面一年更值得写——陌生感消退后需要新的驱动力。
文化碰撞是穿越文的灵魂：穿越者的价值观和当地价值观的摩擦产生好的故事。

先知红利会递减——历史已经被改变，越往后穿越者的预知越不准。
穿越者需要本地盟友，不能永远独行侠——融入比征服更难也更有趣。
最好的穿越文让读者思考：现代人真的比古人聪明吗？还是只是信息多了一点？

## 对话铁律

- 穿越者和当地人的对话有认知差：穿越者的"常识"在当地人听来要么天才要么疯子
- 穿越者偶尔的现代用语泄露是好的叙事工具——但不要每章都出现
- 权贵对话用权力位差体现：上位者不问问题（直接下结论），下位者不说废话（直接表态）
- 穿越者内心的现代吐槽要克制——每章不超过2处，多了会变成相声
- 语言适应要有过程：初期对话理解困难，中期勉强应付，后期才能自如运用

## 反AI写作模式

- 禁止"他利用前世记忆做出了正确决策"——必须展开决策过程和犹豫
- 禁止穿越者对古代一切都"震惊"——适应后的淡然比持续惊叹更真实
- 禁止信息倾倒式介绍历史背景——通过角色的行动和困境自然展示
- 禁止主角对所有古代习俗都完美适应——要有真实的不适和犯错
- 段落中不要把穿越者写得像在考历史试卷——"他知道"不如"他赌了一把"

## 章节钩子

- 适应章结尾：身份伪装出现裂缝 / 偶遇历史人物但对方和记忆中不一样 / 发现蝴蝶效应的第一个迹象
- 布局章结尾：计划推进但被意外的历史事件打断 / 获得资源但引起了不该引起的注意 / 本地盟友提出了意想不到的条件
- 发展章结尾：前世记忆和眼前现实出现矛盾 / 某个"历史注定"的事件没发生 / 发现另一个"不属于这个时代的人"
