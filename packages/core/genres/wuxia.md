## 题材禁忌

- 武功修炼没有过程直接大成
- 内力无限、轻功无距离限制
- 反派排队单挑，不用围攻、暗器、毒药等合理手段
- 江湖规矩时有时无，看剧情需要
- 武功描写沦为玄幻法术
- 用现代格斗术语替代传统武术描写
- 主角侠义精神=无脑救人

## 武功规则

- 武功有明确的层级体系（招式→内功→轻功→暗器各有品级）
- 内力修炼有时间成本，不可一夜大成
- 武功招式有具体名称和可描述的动作，不是光效
- 同等内力下，招式精妙程度决定胜负
- 越级挑战需要合理的条件（地利、对手伤病、奇招）
- 暗器、毒功、机关有明确的克制关系
- 金手指/秘籍四维约束：
  - 能力上限：有天花板，不可无限提升
  - 附加代价：修炼/使用有代价（经脉损伤、走火入魔风险）
  - 触发条件：领悟需要特定条件（师承、奇遇、实战顿悟）
  - 成长路径：循序渐进，不可跳阶

## 语言铁律

- 武功用动作、速度、力量的体感描写，不用数值和光效
- ✗"他的内力达到了第七层" → ✓"剑身还没到，寒意已经刺穿了他三层棉甲"
- 对话要有江湖味但不造作，不是每句话都要"阁下""在下"
- 打斗中的心理活动极简——来不及想的时候不要写一大段分析
- 江湖称号、门派礼仪要统一，不可随意变化
- 环境描写融入武功：竹林中的剑法、悬崖上的轻功、酒馆里的暗器

## 叙事指导

江湖恩怨和侠义情怀是核心驱动力。
侠之大者为国为民，但小侠也有快意恩仇。
人物关系网（师门、门派、帮会、仇家）是故事骨架。

战斗以招式拆解和临场应变为核心，不是数值碾压。
高手对决讲究心理博弈：试探、虚招、诱敌、亮底牌。
群战和单挑要区分写法——群战重配合和乱中求生。

江湖不只有打打杀杀，还有人情练达、世故圆滑。
酒楼、茶馆、客栈是信息交汇和暗流涌动的场所。
武器有重量、有手感、有历史，不是随便捡起来就用。

## 对话铁律

- 江湖人说话有身份：门派弟子规矩多、散人随意、镖师实在、帮会讲规矩
- ✗"阁下可知，在下乃XXX弟子" → ✓亮出暗号/信物/手法，内行人一看便知
- 武林高手之间的客套暗含试探——"久仰"可能是试探深浅
- 市井对话要有烟火气：茶博士的八卦、伙计的吆喝、掌柜的算盘
- 对决前的对话不超过3个回合——废话说多了就不像高手了
- 茶馆/酒楼里的信息传递是武侠特色——听壁脚得到的消息比正式情报更真实

## 反AI写作模式

- 禁止打斗中的长段心理分析：来得及想的用短句（"快！"），来不及想的直接写身体反应
- 禁止所有打斗都写成招式名称接龙：✗"他使出XXX，对方回以YYY" → ✓写动作的速度、轨迹、声音
- 禁止每个配角出场都交代其师承和来历——大部分人只需要"穿青衫的年轻人"
- 禁止用排比渲染武功威力：一个精准的细节比三个比喻更震撼
- 江湖不是真空——客栈要有住客、街道要有路人、山路要有行商

## 章节钩子

- 战斗章结尾：对手临死前说出师承——原来是某门派的人 / 搜身发现令人意外的信物 / 远处有人一直在旁观
- 江湖章结尾：客栈投宿遇到似曾相识的面孔 / 听到一个涉及自身的江湖传言 / 一封来路不明的请帖
- 布局章结尾：联盟达成但某人的眼神不对 / 仇家的行踪有了新线索 / 师门传来急讯
