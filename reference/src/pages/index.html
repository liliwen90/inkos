    📖 AI小说生成器使用教程

    从零开始创作一部AI小说的完整流程

    📑 目录

    1. 新建项目 & 选择分类 2. 配置 LLM（大语言模型） 3. 编写故事创意 & 故事弧线 4. 生成章节大纲 5. 人性化设置 & 风格学习（生成前配置） 6. 生成小说章节 7. 改写与润色工具 8. AI审稿 & 质量评分 9. 章节管理与导出 10. 常见问题

    1. 新建项目 & 选择分类

    进入「提示词管理」页面：

    1. 在「小说分类」中选择你要写的类型（玄幻、言情、都市等14种分类）
    2. 在「项目名称」中输入小说名称（如"星辰大海"），点击新建
    3. 系统会自动加载该分类的默认提示词模板
    4. 你可以根据需要修改「作者角色定义」和「创作规则」
    5. 点击💾 保存提示词
    💡 提示：提示词是AI写作的"灵魂"。作者角色定义决定了AI"是谁"，创作规则决定了AI"怎么写"。花时间调试好提示词，后续生成质量会显著提升。

    2. 配置 LLM（大语言模型）

    进入「LLM配置」页面：

    1. 系统支持 OpenAI 兼容 API 和本地 Ollama 两种模式
    2. 填写你的 API Key、Base URL 和 模型名称
    3. 在「任务路由」中，可以为不同任务指定不同模型：
      • 日常创作：推荐 deepseek-chat（性价比高）
      • 扩写：推荐 deepseek-chat
      • 润色改写：推荐 Claude（文学能力强）
      • 风格分析：推荐 Gemini（分析能力强）
    4. 点击💾 保存路由配置

    3. 编写故事创意 & 故事弧线

    进入「小说生成」页面，在顶部找到故事弧线规划卡片：

    1. 选择项目，输入总章节数
    2. 如果已完成风格学习，弧线编辑器上方会出现 🤖 AI建议弧线，点击即可一键填充
    3. 也可以点击🤖 AI生成弧线按钮，AI会根据你的故事创意规划整本书的结构
    4. 你可以手动编辑弧线 JSON，然后保存
    💡 提示：故事弧线会在每一章生成时自动注入，告诉AI当前章节在全书中的位置和目标。

    4. 生成章节大纲

    在「小说生成」页面的章节大纲生成区域：

    1. 选择小说名称
    2. 填写故事创意/主题——如果已完成风格学习，会有 🤖 AI建议 可直接点击采用
    3. 点击开始生成大纲（每次生成1章大纲）
    4. 点击查看大纲列表可以预览和编辑已有大纲
    5. 重复此步骤生成多章大纲

    5. 人性化设置 & 风格学习（⭐ 推荐第一步就做）

    在「小说生成」页面，展开「🎨 人性化写作设置」卡片：

    1. 选择项目，点击📂 导入TXT小说文件，导入一本或多本风格一致的小说
    2. 点击🧠 开始学习风格，AI会分析风格指纹 + 自动生成全方位创作建议
    3. 学习完成后，所有手填项（故事创意、弧线、角色、世界设定等）都会出现🤖 AI建议选项
    4. 点击AI建议芯片即可一键填充，也可以选"✏️ 自定义输入"手动填写
    5. 人性化参数（视角、时态、节奏等）会被AI自动推荐最佳值，带🤖推荐标记
    💡 提示：这是推荐的第一步！先让AI学习你喜欢的小说风格，然后所有后续步骤都能一键填充，极大简化工作流。没有完整小说时，也可以粘贴片段代替。

    6. 生成小说章节

    在「小说生成」页面的小说生成区域：

    1. 选择项目，点击📊 检测进度确认当前章节
    2. 勾选所需选项：角色状态、世界设定、前面章节（推荐全部勾选）
    3. 选择状态更新模式（连续生成多章时自动使用全自动模式）
    4. 设置生成章节数：设为1逐章生成，设为N则自动连续生成N章
    5. 点击🚀 开始生成
    6. 连续模式下，系统会为每章自动：补生大纲 → 生成内容 → 生成摘要 → 更新角色/世界状态 → 下一章
    7. 随时可以点⏹️ 停止生成终止，已完成的章节不会丢失
    💡 全自动流程：导入TXT → AI学习 → 一键应用全部建议 → 设置生成20章 → 点击开始 → 去喝咖啡 → 回来收获一本小说

    7. 改写与润色工具

    进入「工具箱」页面：

    1. 粘贴要改写的段落
    2. 选择改写模式：润色、生动化、精简、情感深化、对话自然化、五感增强
    3. 五感增强模式可选择具体增强哪些感官维度
    4. 点击🚀 开始改写，结果可直接复制使用

    8. AI审稿 & 质量评分

    在「小说生成」页面底部找到AI审稿区域：

    1. 输入要审稿的章节号
    2. 点击📝 开始审稿
    3. AI会从5个维度评分：AI味、连贯性、角色塑造、文学性、沉浸感
    4. 根据评分反馈，可以用改写工具针对性优化

    9. 章节管理与导出

    在「工具箱」页面：

    1. 章节管理器：预览、编辑、删除已生成的章节
    2. 小说导出：将所有章节合并导出为 TXT 或 HTML 文件

    10. 常见问题

    Q: 生成的章节内容不连贯怎么办？
    A: 确保勾选了"使用前面章节"和"使用角色状态"，并且已经配置了故事弧线。弧线能让AI知道当前章节的全局位置。
    Q: AI味太重怎么办？
    A: 1) 在「小说生成」页面展开「人性化写作设置」，使用「多书风格学习」导入优秀小说训练指纹；2) 调高创意度；3) 生成后使用改写工具逐段优化。
    Q: API调用失败/超时怎么办？
    A: 系统已内置自动重试机制（最多2次重试），如果仍然失败请检查网络或API Key是否有效。
    Q: 可以使用本地模型吗？
    A: 可以。在LLM配置中选择 Ollama，确保本地 Ollama 服务正在运行，然后选择模型即可。
    Q: 推荐的工作流程是什么？
    A: 提示词配置 → LLM配置 → 故事弧线 → 人性化设置 → 逐章生成大纲 → 逐章生成内容 → AI审稿 → 改写润色 → 导出。
